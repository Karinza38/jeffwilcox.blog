A few weeks back at MIX’08 I took along my $150 Aiptek A-HD camera with the intention of shooting some quick video footage while at MIX and using it to start learning Expression Encoder.  It was a success!  Instead of a thorough trip report, I’m going to walk you through the control source download quickly.

In this video I introduce the control source and unit test announcement from MIX, and then walk you through:

  • exploring the Silverlight 2 Beta 1 control source code, unit test and unit test framework download
  • using Visual Studio 2008 to run the control unit tests
  • adding the Exclusive attribute to narrow the test(s) that run in the browser when analyzing tests
  • running the tests in Firefox
  • running the tests in Expression Blend 2.5

I hope you find this useful, I’m planning on screencasting a more in-depth look at the test framework in the future.  I’ll try and enunciate a little better next time, the audio could have been clearer.

Speaking of video, don’t forget that all of the sessions for the MIX conference are online at http://sessions.visitmix.com/.
